## 📢 重要公告

### 114 學年度 APCS 重大變更

:::danger 重要變更
**程式識讀新增 Python 題本**

自 114 學年度（2025年）起，APCS 程式識讀測驗新增 **Python 題本**，考生可選擇：
- **C/C++ 題本**（原有）
- **Python 題本**（新增）

這對 Python 學習者是一大利多！
:::

### 對考生的影響

**優勢**：
- ✅ Python 語法更簡潔易讀
- ✅ 減少語法細節的困擾
- ✅ 更能專注於演算法邏輯

**建議**：
- 選擇你最熟悉的語言
- Python 使用者可以更有信心
- 務必熟悉 Python 的語法細節
