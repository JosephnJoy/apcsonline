# 🚀 快速入門指南

歡迎使用 APCS Online！這份指南將幫助你快速上手。

## 📖 我是學生，想開始學習

### 第一步：了解 APCS
前往 [關於 APCS](/zh/about-apcs) 頁面，了解：
- APCS 是什麼
- 為什麼要考 APCS
- 考試形式與級分制度

### 第二步：評估你的程度

#### 完全新手（沒學過程式）
👉 從 [模組一：奠定基石](/zh/course/module-1/) 開始
- A1: Python 環境與高速 I/O
- A2: 條件判斷與迴圈
- A3: 一維陣列

**預計時間**：4-6 週  
**目標級分**：1-2 級分

#### 有基礎（學過 Python 基礎語法）
👉 複習模組一，然後進入 [模組二：構築核心結構](/zh/course/module-2/)
- B1: 字串操作
- B2: 二維陣列與模擬
- B3: 函式與模組化

**預計時間**：8-10 週（含模組一）  
**目標級分**：3 級分

#### 中階學習者（熟悉基本資料結構）
👉 直接挑戰 [模組三：安裝高速電梯](/zh/course/module-3/)
- C1: 堆疊與佇列
- C2: 遞迴與回溯
- C3: 排序與二分搜尋

**預計時間**：12-16 週（含前兩個模組）  
**目標級分**：4 級分

#### 進階挑戰者（目標 5 級分）
👉 攻克 [模組四：封頂豪華頂層](/zh/course/module-4/)
- D1: 圖論與 DFS、BFS
- D2: 動態規劃
- D3: 分治法與貪心法

**預計時間**：20-24 週（完整課程）  
**目標級分**：5 級分

### 第三步：開始學習

1. **閱讀課程單元**：仔細閱讀概念講解
2. **練習範例程式**：親自動手寫每一個範例
3. **完成 Quiz**：測試你的理解
4. **做推薦習題**：前往 ZeroJudge 提交答案
5. **持續練習**：每天至少 1 小時

### 第四步：實戰練習

前往 [實戰題庫](/zh/problems/)：
- 按級分選擇題目
- 按主題分類練習
- 查看解題建議

## 🎓 我要準備升學

### 不同級分的策略

- **5 級分**：申請頂尖大學 APCS 組

## 💻 技術使用指南

### 設定開發環境

- 前往 [Python 官網](https://www.python.org/) 下載
- 建議版本：Python 3.8 以上

#### 2. 選擇程式編輯器
推薦選項：
- **VS Code**（推薦新手）
- **PyCharm**
- **Sublime Text**

#### 3. 練習平台
- [ZeroJudge](https://zerojudge.tw/)：高中生程式解題系統
- [APCS 官網](https://apcs.csie.ntnu.edu.tw/)：歷屆試題

### 學習工具

#### 使用 sys.stdin（重要！）
從模組一 A1 單元開始就要學會：
```python
import sys
n = int(sys.stdin.readline())
```

#### 測試程式碼
```bash
# 在終端機執行
python your_code.py < input.txt
```

## 🆘 常見問題

### Q: 我沒有程式基礎，能學會嗎？
A: 完全可以！從模組一開始，循序漸進地學習。本課程專為零基礎到進階的學習者設計。

### Q: 需要多久才能達到 4 級分？
A: 依個人基礎與投入時間而異，通常需要 12-16 週的系統學習。

### Q: Python 會不會太慢，無法通過時間限制？
A: 只要使用本課程教的高效能寫法（如 sys.stdin、collections.deque），Python 完全可以通過 APCS 的時間限制。

### Q: 應該選擇 Python 還是 C++？
A: 選擇你最熟悉的語言。Python 語法簡潔，開發快速，但需注意效能；C++ 執行速度快，但語法較複雜。

### Q: 如何知道自己的進度？
A: 完成每個模組的所有單元和習題後，嘗試做該級分的歷屆試題，檢驗學習成效。
