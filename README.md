## 🏢 APCS Python 摩天大樓學習平台

這是一個為 APCS（大學程式設計先修檢測）考生打造的完整學習平台，採用「摩天大樓」的學習比喻，將五個級分目標類比為從地基到頂層的建築工程。

## 🚀 特色功能

- 📚 **完整課程體系**：從基礎到進階，涵蓋 APCS 所有考點
- 🔄 **中英雙語支援**：一鍵切換繁體中文/English
- 💻 **實戰導向**：每個單元包含範例程式、解說、Quiz 和推薦習題
- ⚡ **性能優先**：強調 Python 在競技程式設計中的高效寫法
- 📱 **響應式設計**：支援各種裝置瀏覽

## 📖 課程模組

### 模組一：奠定基石（1-2 級分）
- A1: 競技程式設計師的工具箱：Python 環境與高速 I/O
- A2: 邏輯的骨架：精通條件判斷與迴圈
- A3: 數據的容器：將 List 作為一維陣列

### 模組二：構築核心結構（3 級分）
- B1: 解構文本：進階字串與字元操作
- B2: 超越線性：進階陣列技巧與模擬
- B3: 複雜工程學：函式、模組化與作用域

### 模組三：安裝高速電梯（4 級分）
- C1: 順序與存取：堆疊 (LIFO) 與佇列 (FIFO)
- C2: 探索解空間：遞迴、枚舉與回溯
- C3: 效率的必要性：排序與二分搜尋

### 模組四：封頂豪華頂層（5 級分）
- D1: 探索網路：圖論與 DFS、BFS
- D2: 優化的藝術：動態規劃 (DP)
- D3: 進階演算法策略：分治法與貪心法則

## 📦 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run dev

# 建立生產版本
npm run build

# 預覽生產版本
npm run preview
```
